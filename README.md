# Personal Finance Tracker
 
A simple console-based application for managing personal finances. It allows users to record transactions, review their financial history, and gain insight into spending habits through summaries and category filters.
 
## Features
 
1. **Add Transaction** — Record a new income or expense entry, including details such as amount, category, and date saving transaction to JSON file.
2. **View All Transactions** — Display a complete list of all recorded transactions in chronological order.
3. **View Summary** — Get an overview of total income, total expenses, and net balance.
4. **Filter Transactions By Category** — Narrow down the transaction list to a specific category (e.g., Food, Rent, Utilities) for focused review.
5. **Exit** — Close the application safely.
## Menu
 
```
1. Add Transaction
2. View All Transactions
3. View Summary
4. Filter Transactions By Category
5. Exit
```
 
## Purpose
 
This project is designed to help users track their day-to-day income and expenses in a lightweight, easy-to-use format — ideal for personal budgeting without the overhead of a full finance application.
 
## Tech Stack
 
- **Language:** C#
- **Interface:** Console application
